 Workspace Occupancy Monitoring System

Real-time people detection and occupancy analysis using YOLOv8 and OpenCV.

 Structure

```
backend/  
frontend/  
models/  
data/  
```

 Setup
```
python -m venv venv  
venv\Scripts\activate   # Windows  
source venv/bin/activate # Mac/Linux  
pip install -r requirements.txt  
```

 Run
```
python frontend/app.py  
```

 Features
* People detection
* Live counting
* Video/Webcam input
* Data logging

Tech
Python, OpenCV, YOLOv8, NumPy, Pandas
